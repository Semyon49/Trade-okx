# main.py
Вызов setup_logger перемещен внутри основной функции, чтобы инкапсулировать настройку регистратора в основное выполнение.
Комбинированные операторы импорта для лучшей читаемости.
Вызов load_dotenv перенесен за пределы основной функции, если это нужно сделать один раз для всего скрипта.

Упростил код, в некоторых местах напрямую возвращая значения.

Удалена ненужная переменная (fmt) и интегрирован форматтер непосредственно там, где это необходимо.
